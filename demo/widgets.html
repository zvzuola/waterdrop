下拉菜单

主要用于导航条上的下拉菜单，在下拉按钮的基础上添加‘wd-hover’类。实现与导航条相应的样式。

我是下拉菜单
  • hello，hello
  • second，hello
  • third，hello
  • fourth，hello
 
									
										<span class='wdm-dropdown wd-hover'>
											<a class='wd-dropdown-trigger' href="javascript:;">我是下拉菜单</a>
											<ul class='wd-dropdown-menu wd-target'>
												<li><a href="javascript:;">hello，hello</a></li>
												<li><a href="javascript:;">second，hello</a></li>
												<li><a href="javascript:;">third，hello</a></li>
												<li><a href="javascript:;">fourth，hello</a></li>
											</ul>
										</span>

分页

主要是针对rails实现的分页样式。

一般分页

在rails的分页插件基础上，只需添加‘wdm-pagination-rails’类。

上一页 1 2 3 … 97 98 下一页 
								
									<div class="wdm-pagination-rails">
			                			<div class="pagination" outter_window="0">
			                				<span class="previous_page disabled">上一页</span>
			                				<em class="current">1</em>
			                				<a data-remote="true" href="javascript:;" rel="next">2</a>
			                				<a data-remote="true" href="javascript:;">3</a>
			                				<span class="gap">…</span>
			                				<a data-remote="true" href="javascript:;">97</a>
			                				<a data-remote="true" href="javascript:;">98</a>
			                				<a class="next_page" data-remote="true" href="javascript:;" rel="next">下一页</a>
			                			</div>
			              			</div>

迷你分页

在‘wdm-pagination-rails’类的基础上添加‘wdm-pagination-rails-mini’类可实现迷你的分页样式

上一页 1 2 3 … 97 98 下一页 
								
									<div class="wdm-pagination-rails wdm-pagination-rails-mini">
			                			<div class="pagination" outter_window="0">
			                				<span class="previous_page disabled">上一页</span>
			                				<em class="current">1</em>
			                				<a data-remote="true" href="javascript:;" rel="next">2</a>
			                				<a data-remote="true" href="javascript:;">3</a>
			                				<span class="gap">…</span>
			                				<a data-remote="true" href="javascript:;">97</a>
			                				<a data-remote="true" href="javascript:;">98</a>
			                				<a class="next_page" data-remote="true" href="javascript:;" rel="next">下一页</a>
			                			</div>
			              			</div>

单选框

按照下面实例结构，只需改变input的选中或禁止状态即可实现相应的样式。

								
									<label class='wdm-radio'><input type="radio"><b class='icon-radio'></b>我是单选框</label>
									<label class='wdm-radio'><input type="radio" checked><b class='icon-radio'></b>我是单选框</label>
									<label class='wdm-radio'><input type="radio" checked disabled><b class='icon-radio'></b>我是单选框</label>
									<label class='wdm-radio'><input type="radio" disabled><b class='icon-radio'></b>我是单选框</label>

									<label class='wdm-checkbox'><input type="checkbox"><b class='icon-checkbox'></b>我是多选框</label>

									<label class='wdm-checkbox'><input type="checkbox" disabled><b class='icon-checkbox'></b>我是多选框</label>
