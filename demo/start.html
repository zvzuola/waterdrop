WaterDrop

一些常见的，可重复使用的组件库，如图标，按钮，表格等。

  • 主要文件
  • 色彩
  • 基本模板

主要文件

包含waterdrop.css和waterdrop.min.js文件。

注意：WaterDrop 的所有 JavaScript 插件都依赖 jQuery，因此 jQuery 必须在 WaterDrop 之前引入。

基本模板

							
								<!DOCTYPE html>
								<html lang="en">
									<head>
										<meta charset="UTF-8">
										<title>起步 · waterdrop</title>
										<link rel="stylesheet" href="../css/waterdrop.css">
									</head>
									<body>
										<h1>你好，世界！</h1>
		    							<script type="text/javascript" src="jquery-1.11.1.js"></script>
		    							<script type="text/javascript" src="../dest/waterdrop.min.js"></script>
									</body>
								</html>
